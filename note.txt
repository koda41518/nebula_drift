 Pew Pew Pew - Space Survival Game 🕺🏼

    Un petit jeu arcade de survie dans l’espace 🚀💥
    Réalisé avec Pygame, avec des sprites custom dessinés 👨‍🎨✨
    in progress 🏃‍♂️ -u-

───────────
Assets :
───────────
    Les sprites dessinés 👨‍🎨✨ sur ce site :
    => https://www.pixilart.com/draw. 

    Les images sont placées dans le dossier assets/ (vaisseau, ennemis, bouclier, réparateurs, etc.)🏃‍♀️‍➡️

──────────────────────
Fichiers principaux :
──────────────────────
    main.py
    C’est le fichier principal du jeu.
    => Pour lancer la version  clean  :   "python main.py" 

    mvt-base.py
    Une ancienne version du jeu, plus simple,  avec juste la mécanique de déplacement.
    => Pour tester cette version :       "python mvt-base.py"

────────────────────────────────
Commande pour lancer le jeu :
────────────────────────────────

    Avant de lancer le jeu, installe les dépendances ; 
    dans un env " python3 -m venv env
                  source env/bin/activate"
                   
    pip install -r requirements.txt (numpy - Pygame)
